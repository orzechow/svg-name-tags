SVG Grid Name Tag Tool

Create a printable SVG grid with name tags from a template and a list of names.

How to prepare your SVG template (Inkscape):
  • Create your SVG template in Inkscape.
    Make sure it contains one text element with a placeholder name.
    Do not convert the name text field to paths.
  • Save your template as plain SVG (not as Inkscape SVG).
